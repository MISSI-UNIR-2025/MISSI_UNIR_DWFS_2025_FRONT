# Book Store Front

This is a book store front built to gruop 6 from the [DWFS - UNIR 2026]

## Prerequisites

- Node.js version 22.x
- NPM version 8.x

## Installation

1. Clone the repository

```bash
git clone https://github.com/samuelcalderonhuaccha/DWFS-2026.git
```

2. Install the dependencies

```bash
npm install
```
3. Create a .env file in the root directory and add the following content
```
VITE_API_URL=http://localhost:8081
```


3. Start the application

```bash
npm run dev
```

4. Open the application in your browser

```bash
    http://localhost:5173/
```
